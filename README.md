# TKinter-Music-Player
A simple GUI-based music player built with Python using Tkinter and Pygame, featuring audio playback controls, playlist management, and dynamic song search from a local directory.

## Features
- Play, pause, unpause, and stop songs
- Load and save playlists (.m3u)
- Search songs in real time
- Refresh songs from local folder

## Requirements
Install pygame:
pip install pygame

## How to Run
python main.py

## Notes
- Add your own music files to the `music/` folder
- Supported formats: .mp3, .wav, .ogg
